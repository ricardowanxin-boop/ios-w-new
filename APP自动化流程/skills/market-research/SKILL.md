# Goal
Generate and validate focused iOS app opportunities with commercial potential.

# Workflow
1. Identify narrow user groups.
2. Identify recurring painful situations.
3. Suggest app wedges.
4. Score each opportunity.
5. Eliminate weak ideas.
6. Produce a top 3 shortlist.

# Output files
- /research/briefs/YYYY-MM-DD-opportunity-list.md
- /research/evidence/YYYY-MM-DD-validated-opportunities.md
- /research/competitor-analysis/YYYY-MM-DD-competitor-matrix.md

# Scoring criteria
- pain intensity
- frequency
- willingness to pay
- MVP simplicity
- competition intensity
- launch realism

# Avoid
- vague user groups
- generic productivity ideas
- backend-heavy ideas
- social or marketplace dependence
