# Goal
Turn one validated opportunity into a sharp, narrow, paid iOS product definition.

# Workflow
1. Define target user and trigger moment.
2. Define the painful job to be done.
3. Write the positioning statement.
4. Create the PRD.
5. Create the MVP scope.
6. Define pricing and success criteria.
7. List non-goals explicitly.

# Output files
- /product/positioning/YYYY-MM-DD-positioning.md
- /product/prd/YYYY-MM-DD-prd.md
- /product/prd/YYYY-MM-DD-mvp-scope.md
- /product/pricing/YYYY-MM-DD-pricing-hypothesis.md

# Standards
- one primary use case
- one clear payment reason
- aggressive scope reduction
- explicit non-goals
