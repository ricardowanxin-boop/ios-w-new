# Goal
Build the smallest useful SwiftUI version of the product.

# Workflow
1. Propose the app structure.
2. Define models and persistence.
3. Define state management.
4. Implement core screens.
5. Implement the primary flow.
6. Add essential edge states.
7. Prepare milestone notes.

# Output files
- /app/ios/docs/YYYY-MM-DD-implementation-plan.md
- /app/ios/docs/YYYY-MM-DD-architecture-notes.md

# Standards
- SwiftUI first
- local persistence first
- minimal dependencies
- no speculative infrastructure
- clear comments only where needed
