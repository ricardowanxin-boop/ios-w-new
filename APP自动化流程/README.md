# iOS App Factory

This repository is a Codex-first workflow for taking an iOS app from:
1. market opportunity discovery
2. pain validation
3. product definition
4. UX/UI design
5. SwiftUI implementation
6. QA review
7. App Store launch preparation

## Main folders
- /agents: role prompts for specialized agents
- /skills: reusable workflows Codex can load when needed
- /research: raw opportunity discovery and validation artifacts
- /product: positioning, PRD, MVP scope, pricing
- /design: copy, screens, visual direction
- /app: implementation plans and app code
- /qa: test plans and release blockers
- /launch: App Store, privacy, screenshots, launch plan

## Recommended workflow
1. Run opportunity discovery
2. Validate and eliminate weak ideas
3. Select one product direction
4. Create positioning, PRD, and MVP scope
5. Create UX copy and UI structure
6. Implement in SwiftUI
7. Run QA review
8. Prepare App Store assets and launch plan

## Important rule
Do not start major coding until the chosen direction has:
- evidence summary
- PRD
- MVP scope
- screen inventory
